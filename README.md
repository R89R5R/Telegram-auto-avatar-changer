### ENG
**Avatar Changer for Telegram** is an automated Python script that allows you to automatically change avatars in your Telegram account at a specified interval. The script takes images from the specified folder, sets them as a profile avatar, and moves the processed files to separate folders for the workflow organization.

## Main features
- **Automatic change of avatars** - install new avatars with a configurable interval.
- **File organization** - automatic sorting by folders: source, processed, erroneous.
- **Image preparation** - automatic image optimization and conversion for Telegram.
- **Configurable intervals** - the ability to set any time between avatar shifts.
- **Process logging** - detailed output of statuses and execution statistics.
- **Error handling** - nondisruptive correct handling of problem files.

## Technical features
- **Telegram Bot API** - using the official Telegram API.
- **PIL/Pillow** - image processing and optimization.
- **Asynchronous programming** - efficient work with network requests.
- **Cross-platform** - runs on Windows, macOS, Linux.

## Requirements

- Python 3.7+
- Libraries: `telethon`, `Pillow`
- API keys from Telegram (obtained on my.telegram.org)

## Security

- Using the official Telegram API.
- Setting safe intervals between requests.
- Recommendation to use a test account.
- Local storage of sessions.

## Application

- Personal use for frequent change of avatars.
- Automation of social activities.
- Testing the functionality of the Telegram API.
- Organization of content strategy in the messenger.

## Project Status

**Worker and tested** - the script successfully completes the task of automatically changing avatars in Telegram with the full organization of the file workflow.

### RU
**Avatar Changer for Telegram** - это автоматизированный скрипт на Python, который позволяет автоматически менять аватары в Telegram аккаунте с заданным интервалом. Скрипт берет изображения из указанной папки, устанавливает их как аватар профиля и перемещает обработанные файлы в отдельные папки для организации workflow.

## Основные возможности
- **Автоматическая смена аватаров** - установка новых аватаров с настраиваемым интервалом.
- **Организация файлов** - автоматическая сортировка по папкам: исходные, обработанные, ошибочные.
- **Подготовка изображений** - автоматическая оптимизация и конвертация изображений для Telegram.
- **Настраиваемые интервалы** - возможность установить любое время между сменами аватаров.
- **Логирование процесса** - детальный вывод статусов и статистики выполнения.
- **Обработка ошибок** - корректная обработка проблемных файлов без прерывания работы.

## Технические особенности
- **Telegram Bot API** - использование официального API Telegram.
- **PIL/Pillow** - обработка и оптимизация изображений.
- **Асинхронное программирование** - эффективная работа с сетевыми запросами.
- **Кросс-платформенность** - работает на Windows, macOS, Linux.

## Требования

- Python 3.7+
- Библиотеки: `telethon`, `Pillow`
- API ключи от Telegram (получаются на my.telegram.org)

## Безопасность

- Использование официального Telegram API.
- Настройка безопасных интервалов между запросами.
- Рекомендация использования тестового аккаунта.
- Локальное хранение сессий.

## Применение

- Персональное использование для частой смены аватаров.
- Автоматизация социальных активностей.
- Тестирование функциональности Telegram API.
- Организация контент-стратегии в мессенджере.
